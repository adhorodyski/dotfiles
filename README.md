# dotfiles

Use [Dotbot](https://github.com/anishathalye/dotbot) to install.

## Stack

Terminal: [Ghostty](https://ghostty.org/)

Editor: [Neovim](https://neovim.io/) with [LazyVim](https://www.lazyvim.org/)

Shell: [Zsh](https://formulae.brew.sh/formula/zsh) with [Oh my Zsh](https://ohmyz.sh/) and the [minimal](https://github.com/subnixr/minimal) theme
