# dotfiles

## Prerequisites

[git](https://git-scm.com/) and [dotbot](https://github.com/anishathalye/dotbot).

Clone this repository, then use  `dotbot` to install:

```sh
dotbot -c install.conf.yaml
```

## Platforms

- macOS ([Homebrew](https://brew.sh/))
- Arch (pacman)
- Debian (apt)
